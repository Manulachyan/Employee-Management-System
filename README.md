# Employee Management System

## Description

A desktop-based Employee Management System developed using Java Swing and MySQL with a modular structure to efficiently manage employee records.

## Features

* Add new employees
* Edit existing employee details
* Delete employee records
* View all employees
* Manage employee salary
* Login system for access control

## Project Modules

* AddEmployee.java
* EditEmployee.java
* DeleteEmployee.java
* ViewEmployees.java
* EmployeeSalary.java
* LoginPage.java
* DatabaseConnection.java

## Tech Stack

* Java
* Java Swing
* MySQL

## How to Run

1. Open the project in Eclipse
2. Configure MySQL database connection
3. Run MainMenu.java
